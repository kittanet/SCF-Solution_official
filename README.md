## SCF-Solution

supply chain finance solutions with block chain

## Installation

testrpc
```sh
$ npm install -g ethereumjs-testrpc
```

truffleframework
```sh
$ npm install -g truffle
```

```sh
$ npm install
```

Run app
```sh
$ testrpc
```

```sh
$ truffle compile
$ truffle migrate
```

```sh
$ npm start
```
